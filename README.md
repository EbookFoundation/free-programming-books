## Intro

This list was originally a clone of [StackOverflow - List of Freely Available Programming Books](https://stackoverflow.com/questions/194812/list-of-freely-available-programming-books).

The list was moved to GitHub by [Victor Felder](https://github.com/vhf) for collaborative updating and maintenance.
