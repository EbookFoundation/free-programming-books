# ميثاق سلوك المساهمين

بصفتنا مساهمين ومسؤولين عن صيانة هذا المشروع، ومن أجل تعزيز مجتمع مفتوح ومرحب، فإننا نتعهد باحترام جميع الأشخاص الذين يساهمون من خلال الإبلاغ عن المشكلات، أو نشر طلبات الميزات، أو تحديث الوثائق، أو تقديم طلبات السحب (Pull Requests) أو التصحيحات، وغيرها من الأنشطة.

نحن ملتزمون بجعل المشاركة في هذا المشروع تجربة خالية من المضايقات للجميع، بغض النظر عن مستوى الخبرة، أو النوع الاجتماعي، أو الهوية والتعبير الجنسي، أو التوجه الجنسي، أو الإعاقة، أو المظهر الشخصي، أو حجم الجسم، أو العرق، أو الأصل الإثني، أو العمر، أو الدين، أو الجنسية.
أمثلة على السلوك غير المقبول من قبل المشاركين:

*   استخدام لغة أو صور ذات طابع جنسي.

* الهجمات الشخصية.

* التصيّد (Trolling) أو التعليقات المهينة/المسيئة.

* المضايقات العلنية أو الخاصة.

*  نشر معلومات الآخرين الخاصة، مثل العناوين المادية أو الإلكترونية، دون إذن صريح.

*  أي سلوك آخر غير أخلاقي أو غير مهني.

يتمتع مسؤولو صيانة المشروع بالحق والمسؤولية في إزالة أو تعديل أو رفض التعليقات، والبرمجيات (Commits)، والكود، وتعديلات الويكي، والمشكلات، والمساهمات الأخرى التي لا تتماشى مع ميثاق السلوك هذا، أو حظر أي مساهم مؤقتاً أو دائماً بسبب سلوكيات أخرى يرونها غير لائقة، أو تهديدية، أو هجومية، أو ضارة.

من خلال اعتماد ميثاق السلوك هذا، يلتزم مسؤولو صيانة المشروع بتطبيق هذه المبادئ بإنصاف واتساق على كل جانب من جوانب إدارة هذا المشروع. قد يتم استبعاد مسؤولي الصيانة الذين لا يتبعون ميثاق السلوك أو لا يطبقونه من فريق المشروع بشكل دائم.

ينطبق ميثاق السلوك هذا داخل مساحات المشروع وفي المساحات العامة عندما يمثل الفرد المشروع أو مجتمعه.
الإبلاغ

يمكن الإبلاغ عن حالات السلوك التعسفي أو المسيء أو غير المقبول عبر الاتصال بمسؤول صيانة المشروع على البريد الإلكتروني: victorfelder@gmail.com. ستتم مراجعة جميع الشكاوى والتحقيق فيها، وسيؤدي ذلك إلى رد يعتبر ضرورياً ومناسباً للظروف. يلتزم مسؤولو الصيانة بالحفاظ على سرية هوية مقدم البلاغ.
