# 贡献者行为准则

作为本项目的贡献者和维护者，为了营造开放而友好的社区，我们承诺尊重所有通过报告问题、提出功能请求、更新文档、提交拉取请求或补丁以及其他活动参与贡献的人。

我们致力于让每个人都能在没有骚扰的环境中参与本项目，不论经验水平、性别、性别认同与表达、性取向、残障、个人外貌、体型、种族、族裔、年龄、宗教或国籍。

参与者不可接受的行为示例如下：

* 使用性化语言或图像
* 人身攻击
* 挑衅、侮辱性或贬低性评论
* 公开或私下的骚扰
* 未经明确许可发布他人的私人信息，例如物理地址或电子地址
* 其他不道德或不专业的行为

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 以及其他贡献；对于维护者认为不适当、有威胁、冒犯性或有害的其他行为，也可以暂时或永久禁止相关贡献者参与。

采用本行为准则即表示项目维护者承诺在项目管理的各个方面公平且一致地应用这些原则。不遵守或不执行行为准则的项目维护者可能会被永久移出项目团队。

当个人代表项目或其社区时，本行为准则同时适用于项目空间和公共空间。

如果遇到辱骂、骚扰或其他不可接受的行为，可以通过 victorfelder at gmail.com 联系项目维护者进行报告。所有投诉都会被审查和调查，并会根据具体情况作出必要且适当的回应。维护者有义务对事件报告者的信息保密。
